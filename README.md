 # 🚀 JavaScript Fundamentals - Exercícios de Lógica

Este repositório é o meu "diário de treino" de JavaScript. Aqui reuni todos os exercícios e mini-projetos que desenvolvi para consolidar os conceitos fundamentais da linguagem. Feito em Setembro de 2025.

### ⚖️ Comandos Condicionais (`if/else`, `switch`)
Exercícios focados em tomada de decisão:
- Verificador de desempenho escolar.
- Cálculo de mensalidades por faixa etária.
- Simulador de tarifas de transporte.

### ⚙️ Funções e Operações
Prática de encapsulamento e reutilização de código:
- Calculadora aritmética completa.
- Conversor de moedas em tempo real.
- Calculadora de IMC com diagnóstico de saúde.

### 🔄 Laços de Repetição (`for` e `while`)
Domínio de estruturas de repetição:
- **FOR:** Média de notas da sala, controle de stock de produtos e simulador de sistema de votação.
- **WHILE:** Validação de senhas, somador de produtos e um jogo de "Par ou Ímpar" contra a máquina.

## 🛠️ Tecnologias e Ferramentas
- **Linguagem:** JavaScript (ES6+)
- **Ambiente:** Navegador (Google Chrome / DevTools) e VS Code.
- **Interação:** Uso de `prompt`, `alert` e `console.log`.

## 🚀 Como testar os exercícios
1. Clone este repositório.
2. Como os scripts estão vinculados ao `index.html`, basta abrir o arquivo no seu navegador.
3. Para trocar o exercício que está a ser executado, altere a tag `<script src="...">` no arquivo HTML.
